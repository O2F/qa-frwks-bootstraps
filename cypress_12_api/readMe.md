## Dependencies

To verify the dependencies version use:

```sh
npx npm-check-updates 
```
To automaticaly update:

```sh
npx npm-check-updates -u
```

To install the dependencies just run:

```sh
npm install
```

## How to run

```sh
npm run test
```
